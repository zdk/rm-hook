# rm-hook

Use 'rm' safely

# Install

`./rm-hook.sh install`

# Usage

`rm` to delete things as usual, but safer ＼(◎o◎)／

# In additions::

```
rm -rf directory/  # Backup to trash, then delete
rm --list-trash    # Show trash contents
rm --empty-trash   # Empty trash
```

# Uninstall

`./rm-hook.sh uninstall`
